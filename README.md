# AdaPCL
Evidential Uncertainty Modulated Adaptive Predictive Contrastive Learning for Multimodal Fusion

Code for Evidential Uncertainty Modulated Adaptive Predictive Contrastive Learning for Multimodal Fusion (https://doi.org/10.1145/3805622.3810771) (ICMR 2026)

Data
N24News (https://github.com/billywzh717/n24news) is a multimodal news classification dataset. We adopt the original split method.
TumEmo (https://github.com/YangXiaocui1215/MVAN) is a
large-scale image-text emotion dataset with more than 190,000 instances from
Tumblr.
Fakeddit (https://github.com/entitize/fakeddit) is a large-scale dataset designed for analyzing misleading information on Reddit.
